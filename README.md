# Kai Coffee — Demo navegable

Demo PWA navegable para **Kai Coffee** (cafetería de especialidad).
Pantallas hardcodeadas, sin backend / DB / auth real — solo flujo visual.

## Modos (toggle flotante inferior derecho)
- **Público** — Home, Menú (carrito + pago simulado), Espacios (reserva)
- **Cliente** — Dashboard de lealtad, niveles e historial de puntos
- **Admin** — Panel KPIs, Reservas (confirmar/cancelar), Clientes

## Stack
Next.js 16 · App Router · TypeScript · estilos inline · paleta V.Momentum

```bash
npm install
npm run dev
```
